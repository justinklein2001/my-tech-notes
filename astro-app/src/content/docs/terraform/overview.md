### Why Learn Terraform?
After completing the AWS Certified Solutions Architect - Associate certification, I could see how configuring all cloud instances via a GUI would lead to long-term headaches. Terraform felt like the next logical tool to learn to have all my cloud deployments skills be both **cloud-agnostic** and highly **reproducible**.
